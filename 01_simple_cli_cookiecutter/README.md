# CLI COOKIECUTTTER

A cli cookiecutter for quickly spinning up new projects.

## Install

```sh
brew install cookiecutter

cookiecutter --help
cookiecutter --version
```

## Usage

```sh
mkdir -p ./xx_my_new_project

cd ./xx_my_new_project

cookiecutter $(pwd)/../01_simple_cli_cookiecutter
```
