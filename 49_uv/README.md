# UV

A little example of using UV.  

NOTES:

* It is very quick at installing packages.  

## Install

```sh
just nix
```

## Start

```sh
just lint

just format

just test

just start --test
```

## Created

When installing a fresh UV project.  

```sh
# create a basic skeleton
uv init .

# add packages
uv add pyyaml

# add dev 
uv add --dev flake8-2020 
```
