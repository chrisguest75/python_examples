# README

A repo containing some python examples.  

## Configuration

### pyenv

```sh
# install pyenv version manager
brew install pyenv

# list versions available to install
pyenv install --list

# install a version
pyenv install 3.11.1 

# list local versions installed
pyenv versions
```

## pipenv

```sh
pipenv
```
